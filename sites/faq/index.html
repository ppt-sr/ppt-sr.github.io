    FAQ about GAMEPLAY!

    Can I mod the game to change textures or models?

    Any form of modding the game is not allowed.

    Can I use macros or bind keybinds to something else?

    No, the usage of macros or keybinds is not allowed.

    How do I perform X glitch or strat?

    We have guides that can help you!

    How do I instantly Load in Chapter 2?

    There are Hotkeys in chapter 2 to load and pause:
    p for Loading
    o for Pause.

    FAQ about TECHNICAL STUFF!

    How to set up LiveSplit?

    We recommend Smallant's LiveSplit Tutorial to set up your splits, layout, and more.

    My Hotkeys don't work in-game, how do I fix it?

    If your Hotkeys don't work in-game, go to: Right Click -> Settings -> Global Hotkeys and make sure it's checked.

    How do I set up my autosplitter / load remover?

    We provide guides to set up autosplitters for every chapter. Check the main document for links.

    How do I enable console?

    The console is a Tool called "Universal Unreal Unlocker" or "UUU" / "UUEU" for short download it here:
